visualCaptcha has an innovative solution for people with accessibility needs.

2. Mobile-friendly
It works on most mobile devices.

== Installation ==
Wordpress installation

1. Download visualCaptcha
2. Go to 'Plugins' in the Admin Wordpress section.
3. Click on 'Add New'.
4. Click on 'Upload', choose the file and press install.
5. Activate the plugin through the 'Plugins' menu in WordPress.

Manual Installation

1. Upload the `captcha` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
